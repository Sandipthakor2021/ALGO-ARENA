Minimum Number of Pizzas


Ram is throwing a party for his N friends. There is a pizza store nearby and he wants to buy pizza for his friends. Each pizza has exactly K slices. Ram's friends get sad if one gets more slices than the other. Also, Ram gets sad if there are some pizza slices left. More formally, suppose Ram buys P pizzas, then everyone is happy if and only if there is a way to distribute K⋅P slices between N friends.

You need to find the minimum number of pizzas Ram has to buy to share all the slices among his friends so that none of his friends gets sad. Note that Ram hates pizza and doesn't take any slices.

Input Format

  • First line of input contains T, the number of test cases. Then the test cases follow.
  • Each test case contains two space-separated integers N and K, where N is the number of friends of Ram and K is the number of slices in a pizza.

Output Format

For each testcase, print the minimum number of pizzas Ram has to buy to share among his friends so that none of his friends gets sad.

Constraints

  • 1 ≤ T ≤ 2*105
  • 1 ≤ N,K ≤ 109
  • The sum of N over all test cases does not exceed 2*105
    Subtasks
  • Subtask 1 (100 points): Original constraints

Sample 1:

Input Output
3 1
2 2 2
2 3 2
4 2

Explanation:

  • Test case 1: One pizza has 2 slices. And there are 2 friends. So Ram can just buy one pizza and give one slice to one friend and another slice to another friend.
  • Test case 2: One pizza has 3 slices. And there are 2 friends. So Ram can't share one pizza without being left out with a slice. So he needs to buy at least 2 pizzas. And if he buys 2 pizzas, he can give 3 slices to one friend and 3 to another. So the minimum number of pizzas Ram needs to buy is equal to 2.
  • Test case 3: One pizza has 2 slices. And there are 4 friends. So Ram can't share one pizza among the 4 friends. So he needs to buy at least 2 pizzas. And if he buys 2 pizzas, he can give 1 slice to each of his friends. So the minimum number of pizzas Ram needs to buy is equal to 2.
